forms


< form action="index.php" method="POST" enctype="multipart/form-data">                                         //form-> Creates a form where users can enter data
                                                                                                                 action="index.php" → Tells the browser where to send the form data after clicking Submit.
                                                                                                                                      Send all form data to index.php for processing.
                                                                                                                 method="POST" → Specifies HOW data is sent.
                                                                                                                 | Method | Data Visible? | Used For         |
                                                                                                                 | ------ | ------------- | ---------------- |
                                                                                                                 | GET    | Yes (in URL)  | Searching        |
                                                                                                                 | POST   | No            | Passwords, forms |

                                                                                                                 enctype="multipart/form-data" → Required for file uploads.
                                                                                                                 Needed when uploading files.


                                                                                                                  Example:

                                                                                                                  < input type="file">

                                                                                                                  Without this:

                                                                                                                  files will NOT upload properly

                                                                                                                  With this:

                                                                                                                  browser sends file data correctly

                                                                                                                  Why "multipart/form-data"?

                                                                                                                  Normally forms send plain text.

                                                                                                                  But files contain:

                                                                                                                  images
                                                                                                                  PDFs
                                                                                                                  videos
                                                                                                                  binary data

                                                                                                                  So browser uses special format called:

                                                                                                                  multipart/form-data

                                                                                                                  to send both:

                                                                                                                  text fields and file data together.
